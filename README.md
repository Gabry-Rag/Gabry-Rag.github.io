## 🛠️ Deploy su GitHub Pages
Il deploy è gestito in modo completamente automatico da GitHub Actions.
Ogni volta che viene fatto un `push` sul branch `main`, l'action (configurata in `.github/workflows/deploy.yml`) esegue la build e pubblica il sito.

**Note sulle impostazioni del Repository:**
Affinché GitHub Pages funzioni correttamente, assicurati che nelle impostazioni del repo (Settings -> Pages -> Build and deployment) la **Source** sia impostata su **GitHub Actions**.
